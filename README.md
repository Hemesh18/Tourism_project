# Tourism_project
Project Overview

This Tourism Website is designed to showcase popular travel destinations, local culture, attractions, and travel services. The website aims to promote tourism by providing users with an easy-to-navigate platform to explore destinations, plan trips, and discover experiences.
Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

JavaScript – Interactivity and dynamic content
