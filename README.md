#  ADVENTURE - Tourism Website

##  Overview

**ADVENTURE** is a dynamic tourism website designed to help users explore beautiful destinations, discover travel events, and connect for inquiries.
The project combines an attractive frontend with a PHP & MySQL backend to deliver a complete web experience.

---

## ✨ Features

* 🏔️ Explore different travel destinations
* 📅 View upcoming events and tours
* 📩 Contact form to connect with users
* 🔐 User authentication (Login & Signup)
* 🌐 Fully responsive design
* 🎨 Modern UI with animations and dark mode

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Tools:** XAMPP / phpMyAdmin

---

## 📂 Project Structure

```
ADVENTURE/
│
├── index.php
├── login.php
├── signup.php
├── contact.php
├── config.php
├── css/
├── js/
├── img/
└── database/
```

### 2️⃣ Move project to XAMPP

```
C:\xampp\htdocs\adventure
```

### 3️⃣ Start Server

* Start **Apache**
* Start **MySQL**

### 4️⃣ Import Database

* Open phpMyAdmin
* Create database: `tourism_db`
* Import SQL file

### 5️⃣ Run Project

```
http://localhost/adventure/index.php
```

---

## 🚀 Future Enhancements

* 🧑‍💼 Admin dashboard to manage tours
* 💳 Online booking system
* ⭐ User reviews & ratings
* 🔍 Search and filter destinations
* 📱 Mobile app integration
